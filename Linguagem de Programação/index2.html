     Com toda essa demanda por conteúdo, foi proposto pelos professores a criação de um pequeno site de receitas para aplicar os conceitos aprendidos em HTML. Em seguida, foi proposto desenvolver um portfólio pessoal, que incluiria um pequeno currículo, utilizando as tecnologias de marcação de texto, estilo e programação.

     Estudando JavaScript, aprendemos a diferença entre JavaScript e ECMAScript, exploramos a utilidade do Node.js, a estrutura de repetição e os vetores. Com base nesse conhecimento, realizamos alguns exercícios. Por fim, utilizando HTML, CSS e JavaScript, desenvolvemos uma calculadora de imposto de renda.
