  • Ciência de Dados

     Em Ciência de Dados iniciamos a linguagem de programação Phyton, utilizando a plataforma Google Colab e a biblioteca Panda. Com isso importamos uma biblioteca de dados do github chamado Titanic, onde conten todas as infotmações dos passageiros do navio naufragado Titanic. Depois começamos a aplicar o processo de amostragem para fazer algumas análises estatísticas. Com alguns conceitos de amostragem fizemos um exercício com os dados dos alunos da nossa classe de ADS.
